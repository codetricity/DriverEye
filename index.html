THETA S 360 Camera

44mm (W) x 130mm (H) x 22.9mm (17.9mm *) (D) Approx. 125g

  • Component: Camera
  • Cost: $350 one-time purchase cost
  • Function: Record 360 degree images and video

Awesome Camera API

Open API, compatible with Google Open Spherical Camera (OSC) API. Easy and friendly HTTP GET and POST calls over WiFi. Take pictures and video. Download media. Adjust options.

Get started using the API with Unofficial API GUIDE

  • Component: API
  • Cost: Free
  • Function: Control camera over WiFi. Take 360 videos and images. Download, delete, manage media.

Mobile Application

Simple mobile application to take pictures and upload to the site

  • Component: Mobile app controller
  • Cost: Free and open source
  • Function: Use mobile phone to control camera and upload to external site

360 Image Viewer

360 viewer using either THETA360 business site or JavaScript open source viewer

  • Component: Viewer
  • Cost: Free and paid options available
  • Function: Display 360 image in a web page with navigation

Hardware Options

Control the camera from inexpensive devices such as the Raspberry Pi

  • Component: Camera Controller
  • Cost: Software is free. The Raspberry Pi 3 with WiFi is $35. Buy your own.
  • Function: Standalone, single purpose device to control camera

Optional Touchscreen

Example source code to build touchscreen to control camera

  • Component: Camera Controller
  • Cost: Software is free and open source. Touchscreen is about $60. Buy your own.
  • Function: Control camera on standalone, specialized device or standard tablet
